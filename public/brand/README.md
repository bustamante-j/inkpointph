Put InkPoint brand assets here.

Recommended locations:

- Logo files: `public/brand/logo/`
- Mascot files: `public/brand/mascot/`

Suggested filenames:

- `public/brand/logo/inkpoint-logo.png`
- `public/brand/logo/inkpoint-logo.svg`
- `public/brand/mascot/inkpoint-mascot.png`
- `public/brand/mascot/inkpoint-mascot.webp`

Files in `public` are served from the site root. For example:

- `public/brand/logo/inkpoint-logo.png` is available at `/brand/logo/inkpoint-logo.png`
- `public/brand/mascot/inkpoint-mascot.png` is available at `/brand/mascot/inkpoint-mascot.png`

Use SVG for logos when possible. Use PNG or WebP for mascot art.
